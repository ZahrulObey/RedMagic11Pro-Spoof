## 📥 Installation

1. Download file `.zip` di bagian **Releases**
2. Buka **Magisk** → **Modules** → **Install from storage**
3. Pilih file ZIP yang sudah didownload
4. Tunggu proses instalasi selesai
5. **Reboot** perangkat
6. Done! ✅

---

## 🗑️ Uninstall

Buka **Magisk → Modules** → tap tombol hapus di modul ini → Reboot.
